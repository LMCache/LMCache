retrieve(tokens: Tensor) → Tuple[Tuple[Tuple[Tensor, Tensor], ...], int][source]#

Retrieve the KV cache of the tokens from the cache engine. The retrieved KV cache should be a prefix of the input tokens.

Input:

tokens: the input tokens, with shape [seq_len]

format: either ‘huggingface’ or ‘vllm’

For huggingface, it should have the shape of [num_heads, num_tokens, head_size]

For vllm, it should have the shape of [num_tokens, num_heads, head_size]

Output:
kv_tensors: the kv cache of the tokens, in the format of nested

tuples. Will be an empty tuple if no kv cache is retrieved.

num_tokens: the number of tokens in the kv cache

store(tokens: Tensor, kv_tensors_raw: Tuple[Tuple[Tensor, Tensor], ...], skip_existing=True, blocking=True) → None[source]#

Store the KV cache of the tokens into the cache engine.

Input:

tokens: the input tokens, with shape [seq_len] kv_tensors_raw: the kv cache of the tokens, in the format of nested tuples

format: either ‘huggingface’ or ‘vllm’

For huggingface, it should have the shape of [num_heads, num_tokens, head_size]

For vllm, it should have the shape of [num_tokens, num_heads, head_size]

Returns:

None

Note:

The KV cache should NOT have the “batch” dimension.
